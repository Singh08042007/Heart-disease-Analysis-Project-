# Heart Disease Risk Analysis Dashboard

An interactive dashboard that provides comprehensive insights and predictive analysis for cardiovascular health risks based on multifaceted data points. This dashboard features a modern, clean UI, white-themed analytics, and an integrated Tableau visualizer.
